#ミニテラ

## サイト概要
植物の水やりリマインドアプリケーション。
植物の種類ごとに水やりの頻度と量を管理する。

### サイトテーマ
観葉植物の種類に応じた水やり日数をリマインド

### テーマを選んだ理由
手軽に始められる趣味や部屋のインテリアとして観葉植物が人気ですが、育成の知識が無いと植物を枯らしてしまうことも少なく無いでしょう。
実際に私も観葉植物を育てていましたが、水のやりすぎで枯らしてしまった経験があります。
知識がなくても育成のガイドとなるアプリがあれば育てやすいと思いこのテーマにしました。


### ターゲットユーザ
植物育成をしている人。
育成知識があまりない人。

### 主な利用シーン
前回の水やりを忘れてしまった時。
水やりの頻度がわからない時。
